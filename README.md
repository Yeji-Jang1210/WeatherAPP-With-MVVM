# WeatherAPP-With-MVVM
 "WeatherApp"을 클론코딩한 프로젝트로, MVVM 패턴과 SwiftUI를 활용하여 날씨 정보를 표시하는 애플리케이션을 개발했습니다.

# Skills
- SwiftUI
- MVVM Pattern
- OpenWeatherMap: Weather API
- Kingfisher
- CoreData
